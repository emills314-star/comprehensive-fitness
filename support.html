Comprehensive Fitness tracks workouts locally and provides transparent strength progression recommendations.

Common Help

If data seems missing, check that you are using the same browser or installed app. Workout data is stored locally on the device.

Use Settings to export a JSON backup before changing devices or clearing browser/app data.

Install on iPhone

  1. Open the app URL in Safari.
  2. Tap the Share button.
  3. Choose Add to Home Screen.
  4. Enable Open as Web App when that option is shown.
  5. Name it Comprehensive Fitness and tap Add.

After installation, open Settings in the app and complete iPhone app setup to enable and test rest notifications. The app cannot override Silent Mode, Focus, or iPhone notification settings.

After the first successful load, the app shell can reopen offline. Active workout data is stored in IndexedDB for that installed web app/site.
